### What My Project Is About

This week was my first week at ARC! My project is based around S2FFT, a Python package for working with data on the sphere. This is really useful in areas where the data naturally sits on a round surface, like climate modelling, astronomy, geophysics, computer graphics, and machine learning.

The parts that interest me most are using GPU acceleration to speed up these computations, exploring how libraries like JAX can make the algorithms differentiable, and seeing how these tools can be applied to real examples such as simulations.
